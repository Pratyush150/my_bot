# 🚀 ROS2 Motor Control & Simulation Project

## 📌 Overview

This project is a **ROS2-based robotic system** that integrates:

* 🧠 High-level control using ROS2 nodes
* 🎮 A GUI for real-time motor control
* ⚙️ Serial communication with motor hardware
* 🔄 Encoder-based feedback for closed-loop control

It demonstrates how software (ROS2 + Python) interacts with hardware (motors + encoders) — a **core concept in robotics and autonomous systems**.

---

## 🧩 System Architecture

```
GUI (MotorGui)
   ↓
ROS2 Topic (motor_command)
   ↓
Driver Node (MotorDriver)
   ↓
Serial Communication (USB)
   ↓
Motor Controller (Arduino / MCU)
   ↓
Motors + Encoders
   ↓
Feedback (encoder_vals, motor_vels)
   ↓
GUI Updates
```

## 🚀 How to Run

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd ros2_WS
```

---

### 2️⃣ Build the workspace

```bash
colcon build
```

---

### 3️⃣ Source the workspace

```bash
source install/setup.bash
```

---

### 4️⃣ Run nodes

#### Start motor driver:

```bash
ros2 run serial_motor_demo driver
```

#### Start GUI:

```bash
ros2 run serial_motor_demo gui
```

## ⚡ How It Works

1. User sets motor speed via GUI
2. Command is published to `motor_command` topic
3. Driver node converts it into serial commands
4. Arduino executes motor movement
5. Encoders generate feedback
6. Driver calculates speed & publishes data
7. GUI updates in real-time
